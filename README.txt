This folder contains a 360 interactive web application generated by CloudPano, available at http://www.cloudpano.com.

Usage
-----

The main application file is `index.html`. If you were to open index.html in your browser on your local computer, 
it will not load due to browser security policies. YOU MUST UPLOAD THIS PACKAGE TO A WEB SERVER FOR IT TO WORK.
Browsers prevent loading images from the local file system.

To properly view the application it is necessary to use a web server.

If you want to view this application without an internet connection, you may do so. Just run a local webserver on your local computer.
Check out our facebook group - Virtual Tour Profit - and ask for more information!

Deployment
----------

To deploy the application, upload the contents of the folder into
a hosting service.
